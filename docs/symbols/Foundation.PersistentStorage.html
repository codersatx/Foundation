Class Detail
Foundation.PersistentStorage()
Dictionary-based persistent storage. As this class relies on Titanium.App.Properties, for each value Foundation will store an extra value to keep reference of the value's type. This allows you to get and set values without worrying of data types and performances.

{mixed} get(key, includeKey)
Gets one or more values safely.
Parameters:
{mixed} key
The key to remove. If string, the key will be removed, if present. You can provide a regex that will delete all the matching keys.
{boolean} includeKey Optional, Default: false
True to return an object with two properties: key (the key for this value) and value (the value).
Returns:
{mixed} If you provided a string key:
  • If the key is present, the value will be returned.
  • If the key is present and includeKey is true, an object will be returned (see the description for includeKey).
  • If the key is not present, null will be returned.
If you provided a regex:
  • If the regex matches one or more keys, an array of values will be returned.
  • If the regex matches one or more keys and includeKey is true, an array of objects will be returned (see the description for includeKey).
  • If the regex doesn't match any key, an empty array will be returned.
